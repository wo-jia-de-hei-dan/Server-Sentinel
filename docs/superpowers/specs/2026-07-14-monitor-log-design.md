# 本地检测日志

## 目标

让用户在手机内查看每次手动或自动检测的历史结果，而不上传任何数据。

## 交互

- 首页标题行新增日志入口。
- 日志页按时间倒序显示最近 100 次检测。
- 每条记录显示检测时间、来源（手动或自动）、在线数量，以及每个启用服务器的名称、在线状态、延迟或失败原因。
- 日志为空时显示“暂无检测记录”。

## 数据与保留

- 日志通过现有 DataStore 存在本机。
- 手动刷新和 WorkManager 的 15 分钟自动检测都写入同一日志。
- 写入第 101 条时删除最旧记录；不提供云同步或导出。

## 验证

- 单元测试验证日志 JSON 往返和 100 条上限。
- 构建 debug APK，并复制到用户桌面。
