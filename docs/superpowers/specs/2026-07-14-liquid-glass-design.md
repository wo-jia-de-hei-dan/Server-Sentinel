# 深色 Liquid Glass 界面

## 目标

以原生 Jetpack Compose 复刻 Liquid Glass 的视觉语言，让 Server Sentinel 的主页、设置页和检测日志页呈现深色毛玻璃效果。

## 视觉规则

- 保留现有蓝色 Minecraft 夜景背景。
- 主要卡片使用半透明深蓝填充，背景景物可隐约透出。
- 卡片边缘使用低透明度青蓝高光描边，并使用轻微阴影提升层次。
- 概览、服务器状态、日志和设置卡片使用统一 22dp 圆角。
- 圆形图标按钮使用独立玻璃底座，并保留 Material 按压反馈。
- 正文和状态颜色维持现有高对比度，避免玻璃效果降低可读性。

## 实现边界

- 仅使用 Compose 的透明色、描边、阴影和现有背景图；不引入 React 或 Web 依赖。
- 不做高耗电的实时折射或持续动画。
- 监控、通知、配置与日志数据逻辑保持不变。

## 验证

- 编译全部单元测试和 debug APK。
- 在已连接设备可用时验证主界面与日志页；无设备时交付桌面 APK 并说明限制。
